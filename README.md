This repository contains a Jupyter Notebook that demonstrates how to perform instance segmentation using deep learning. It uses PyTorch and Torchvision to fine-tune a pre-trained Mask R-CNN model on a subset of the Oxford-IIIT Pet dataset, allowing the model to accurately detect cats and dogs and separate them from their backgrounds.
